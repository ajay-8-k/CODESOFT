# EduGrade - Advanced Student Grade Calculator

## Project Description
EduGrade is a Java-based console application that helps students calculate their grades easily. It takes marks for 5 subjects, calculates total marks, average percentage, and assigns grades automatically. This project is designed to be unique with extra features like performance analysis, topper comparison, and report generation.

## Features
- Add student name and roll number
- Enter marks for 5 subjects
- Calculate total marks
- Calculate average percentage
- Assign grades automatically
- Display highest and lowest marks
- Subject ranking system
- Pass/Fail status for each subject
- Performance feedback system
- Scholarship eligibility checker
- Attendance eligibility checker
- Improvement prediction for next grade
- Save student report card into text file
- Compare multiple students and find topper
- Menu-driven console program

## Grade System
- 90–100 = A+
- 80–89 = A
- 70–79 = B
- 60–69 = C
- 50–59 = D
- Below 50 = Fail

## Unique Features
This project is different from normal grade calculators because it includes:
- Student comparison system
- File-based report storage
- Scholarship checker
- Improvement suggestions
- Topper leaderboard
- Attendance eligibility
