# Student Management System

A simple and efficient **Student Management System** built using **Core Java**.  
This project helps manage student records with file handling for data storage.

## Features
- Add new student details
- Remove student records
- Search student by ID or Name
- Update student information
- Display all student records
- Sort students by marks
- Find top performer
- Show students with low attendance
- Generate student report
- Save and load data using file handling

## Technologies Used
- Core Java
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling
- Exception Handling
