# ATM Interface System (Java)

## Description
A console-based ATM Interface System developed in Java as part of the CodSoft Internship Program. The application simulates basic ATM operations such as user authentication, balance inquiry, cash deposit, cash withdrawal, and transaction history management.

## Features

- Secure PIN-based login
- Balance Inquiry
- Cash Deposit
- Cash Withdrawal
- Transaction History
- Input Validation
- Menu-Driven Interface
- Object-Oriented Programming Concepts

## Technologies Used

- Java
- OOP (Object-Oriented Programming)
- Console-Based User Interface

## Project Structure

```
ATM.java

Main.java
```

```bash

```

## Functionalities

### Login
Users must enter the correct PIN to access ATM services.

### Check Balance
Displays the current account balance.

### Deposit Money
Allows users to deposit money into their account.

### Withdraw Money
Allows users to withdraw money if sufficient balance is available.

### Transaction History
Displays all deposits and withdrawals performed during the session.
